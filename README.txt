# Invitación Digital Premium — Lucía Castro (XV) — V2

✅ Esta versión funciona incluso si abres `index.html` con doble clic (sin servidor),
porque incluye el `config` embebido dentro del HTML.

## Reemplazar fotos
En `assets/photos/` reemplaza (mismo nombre):
- hero.png
- editorial.png
- gallery-1.png
- gallery-2.png
- gallery-3.png

## Reemplazar música
Reemplaza `assets/music.mp3`

## Probar con servidor (recomendado para todo)
En terminal dentro de la carpeta:
python -m http.server 8000
y abre http://localhost:8000
